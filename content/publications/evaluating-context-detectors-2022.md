abstract: "We categorize object detectors into three groups based on context usage: those with no context (RCNN), partial context (two-stage methods), and full context (single-stage methods). We systematically evaluate the effect of context for each deep detector category and create controlled datasets to test how detectors leverage scene information. We find that single-stage and two-stage detectors utilize context through their large receptive fields."
