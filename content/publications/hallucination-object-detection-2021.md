abstract: "We demonstrate that object detectors can incorrectly identify non-existent objects with precise localization. We address this in visual part verification tasks - determining whether object components are present or absent. We introduce DelftBikes, a new dataset containing 10,000 bicycle photographs with 22 annotated parts per image, including explicit labels for missing or intact parts. We evaluate popular object detectors on this benchmark."
