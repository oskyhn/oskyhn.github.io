abstract: "We address the challenge that obtaining annotated segments for untrimmed videos is time consuming, expensive and error-prone. We propose to use just a single key frame label for each action instance instead of inherently imprecise start and end frames. To tackle class imbalance, we evaluate a Positive-Unlabeled algorithm (PU-learning) and demonstrate on THUMOS'14 and ActivityNet that using a single key frame label gives good results while being significantly faster to annotate."
